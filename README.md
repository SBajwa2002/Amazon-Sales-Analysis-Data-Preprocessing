🛒 Project Title

Amazon Sales Analysis & Data Preprocessing

📖 Overview

This project focuses on analyzing an Amazon sales dataset to extract meaningful insights and understand sales patterns. The dataset contains information about product pricing, discounts, quantity sold, revenue, customer regions, and payment methods.

The main goal of this project is to clean, preprocess, and transform raw data into a structured format, followed by data analysis and visualization to support data-driven decision-making.

Amazon sales datasets are widely used in data science projects to understand customer behavior, product performance, and revenue trends in e-commerce systems .

🎯 Objectives
Perform data cleaning and preprocessing
Handle missing values and duplicate data
Apply One-Hot Encoding on categorical variables
Normalize numerical features
Perform feature engineering
Analyze sales trends and patterns
Visualize data using different graphs

🧹 Data Preprocessing

Data preprocessing is an important step in any data analysis project. It ensures that the dataset is clean and ready for analysis.

In this project, the following preprocessing steps were performed:

Handling missing values using mean and mode
Removing duplicate records
Converting categorical variables into numerical format using One-Hot Encoding
Normalizing data using MinMaxScaler

These steps are essential because real-world datasets often contain incomplete or inconsistent data that can affect analysis results .

🧠 Feature Engineering

Feature engineering was used to create new meaningful variables from existing data.

Examples:

Total_Sales = price × quantity_sold
Estimated_Profit = Total_Sales × 0.2

Feature engineering helps improve analysis by creating more informative features from raw data .

📊 Data Analysis & Visualization

Different types of visualizations were used to understand the dataset:

📊 Bar Graph → Compare top-performing products
📈 Line Graph → Analyze sales trends
📉 Histogram → Understand data distribution
🔥 Heatmap → Identify relationships between features
📌 Subplots → Combine multiple analyses in one view

Visualization plays a key role in identifying trends, patterns, and relationships in data .

📈 Key Insights
High-revenue products contribute significantly to total sales
Discounts impact customer purchasing behavior
Sales distribution shows variation across products
Certain features like price and quantity have strong relationships
